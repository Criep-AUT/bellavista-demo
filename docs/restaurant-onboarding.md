# RestaurantOS Betreiber-Onboarding

Kompakte Gesprächs- und Datenerfassungscheckliste. Angaben erst nach ausdrücklicher Bestätigung als freigegeben markieren.

## A. Stammdaten

- [ ] offizieller Restaurantname
- [ ] Firmenname und Rechtsform
- [ ] Betreiber/Inhaber
- [ ] vollständige Adresse
- [ ] Telefonnummer
- [ ] E-Mail-Adresse
- [ ] bestehende Domain
- [ ] gewünschte Domain

## B. Branding

- [ ] offizielles Logo in geeigneter Qualität
- [ ] Markenfarben
- [ ] gewünschter Stil und Beispiele
- [ ] vorhandene Schriftarten und Nutzungslizenzen
- [ ] Freigabe des Bildmaterials

## C. Restaurantinformationen

- [ ] bestätigte Kurz- und Langbeschreibung
- [ ] Küchenrichtung und Besonderheiten
- [ ] Zielgruppe
- [ ] Sitzplätze innen/außen
- [ ] Außenbereich oder Terrasse
- [ ] Barrierefreiheit
- [ ] Parkmöglichkeiten und Anreisehinweise

## D. Öffnungszeiten

- [ ] reguläre Öffnungszeiten
- [ ] Küchenzeiten
- [ ] Ruhetage
- [ ] Feiertagsregelung
- [ ] Betriebsurlaub und kurzfristige Änderungen

## E. Speisekarte

- [ ] vollständige Speisekarte
- [ ] aktuelle Preise
- [ ] Allergene und Kennzeichnungen
- [ ] Signature-Gerichte
- [ ] saisonale Gerichte
- [ ] Getränke
- [ ] Tages- oder Wochenangebote

## F. Reservierungen

- [ ] Reservierungen erwünscht: ja/nein
- [ ] bevorzugter Kanal
- [ ] Telefon
- [ ] E-Mail
- [ ] WhatsApp
- [ ] späteres Online-System
- [ ] maximale Gruppengröße
- [ ] gewünschte Vorlaufzeit

## G. Bilder

- [ ] Logo

- [ ] Innenraum
- [ ] Team
- [ ] Speisen und Getränke
- [ ] Terrasse/Außenbereich
